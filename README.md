# victor-skills-marketplace

Claude Code plugin marketplace for custom skills.

For Codex, use the local plugin bundle in the sibling `victor-claude-skills` repo at `plugins/victor-codex-skills/`. Codex currently fits better with repo-local skills or local plugin manifests than with a separate hosted marketplace repo.

## Available Plugins

- **victor-claude-skills** — deep-implement, mega-review, grill-me, cross-examine, long-form-article

## Usage

```
/plugin marketplace add victorblaga/victor-skills-marketplace
/plugin install victor-claude-skills
/reload-plugins
```
